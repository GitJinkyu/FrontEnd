자바스크립트 개요

1. 자바스크립트 사용법

inlie(인라인)

internal(내부)

external(외부)

    별도의 .js 파일로 작성해서 html 파일에서 가져다 사용하는 방법
    자바스크립트의 양이 많ㅇ거나 중복되는 코드가 있을 경우
    자바스크립트 코드 부분을 별도의외부 파일로 작성한다.

2. 데이터 입출력

1) 데이터 입력

window.confirm()
            -어떤 질문에 대해 "예/아니오"의 결과를 얻을 때 사용한다
            -확인 버튼과 취소 버튼이 나타나며 확인 버튼 클릭시 true
                취소버튼 클릭시 false를 리턴한다
    
windows.prompt()
                -텍스트 필드,확인,취소 버튼이 있는 대화 상자를 띄운다
                -입력한 메세지 내용을 리턴값으로 돌려받는다

2) 데이터 출력

document.write()
                -브라우저 화면상의 페이지에 직접 값을 출력한다
    
innerHTML 속성으로 요소 내용 변경
자바스크립트에서 태그 요소의 값을 읽거나 변경할 때 innerHTML 속성을 사용한다. 
console.log()
            -개발자 도구의 콘솔 화면에 출력하고자 할 떄 사용한다
                (디버깅할때 주로 사용함)
