2. 이벤트가 발생한 요소 객체에 접근하는 방법

이벤트가 발생한 요소 객체는 내부적으로 이벤트 발생 정보를 담고 있는 객체를 매개변수로 전달한다
이벤트 객체를 통해서 해당 요소 객체에 접근이 가능하고 콜백 함수의 종류에 따라 this로 접근이 가능하다
콜백함수 : 다른코드의 인수로서 넘겨주는 실행가능한 코드

3. 태그별 기본적으로 가지고 있는 이벤트 제거

기본 이벤트를 가지고 있는 요소
a: 클릭 시 href 속성에 제시되어 있는 페이지를 요청하는 기본 이벤트 보유
submit: 클릭 시 입력 양식 제출 후 페이지를 다시 로딩하는 기본 이벤트 보유
기본 이벤트가 있는 요소에 onclick 이벤트를 설정하면 이 코드가 우선적으로 먼저 실행되고 그 다음에 기본 이벤트가 실행된다
태그에 기본적으로 설정되어 있는 이벤트를 제거하기 위해서는 이벤트 핸들러의 return 값을 false로 하면 된다
